make_leader_fasta
=================

.. argparse::
   :filename: ../src/scripts/make_leader_fasta.py
   :func: get_parser
   :prog: make_leader_fasta

Notes
-----

This command will create a FASTA file containing only the leader sequences from a germline reference set in `AIRR-C MIAIRR format <https://docs.airr-community.org/en/latest/datarep/miairr.html>`_.
